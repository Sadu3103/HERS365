## Your Setup Checklist

### Codebases
- [ ] HERS365-real — clone from GitHub, ask Samuel for repo access. React + TypeScript + Vite. Coach portal + athlete portal.

### MCP Servers to Activate

> **Note:** Samuel uses Claude Code. If you're on **Antigravity**, MCP setup works the same way — check Antigravity's MCP settings panel to add each server below.

- [ ] **Claude in Chrome** — lets Claude read and drive your browser (click, fill forms, run JS). Install the Claude Chrome extension, then connect it.
- [ ] **Computer Use** — full desktop control (screenshots, mouse, keyboard). Built into Claude Code / Antigravity; request access per session.
- [ ] **Google Calendar** — create, update, and query calendar events. Authenticate via the MCP plugin OAuth flow.
- [ ] **GitHub** — open PRs, check CI, review diffs without leaving Claude. Authenticate via the GitHub MCP plugin.
- [ ] **iMessage** — read and reply to iMessages from Claude. Requires the iMessage plugin + allowlist setup via `/imessage:access`.
- [ ] **Gmail** — search and draft emails. Authenticate via the Gmail MCP plugin OAuth flow.

### Skills to Know About
- `/using-superpowers` — run this first. Loads the index of every installed skill so you know what's available.
- `/model` — switch between Claude models mid-session (sonnet for speed, opus for hard problems).
- `/compact` — compress context when you're deep in a long session. Run it before context gets bloated.
- `/fulltriage` — full weekly triage: reads Obsidian notes, calendar, GitHub PRs, and email, then writes a prioritized action list.
- `/dailyplan` — builds a focused daily plan. Good for morning standup prep.
